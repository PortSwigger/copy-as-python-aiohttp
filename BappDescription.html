About

A Burp extension to generate async Python code from HTTP requests.
This extension generates different flavors of scripts (e.g. with/without session, with/without main function).
The resulting codes have been tested with Python 3.11.2 but they should work with other versions as well.

Features

  • Generate individual async functions from requests
  • Combine multiple requests into a single session
  • Generate async script for password spraying attacks
